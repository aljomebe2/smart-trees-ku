# Smart Trees — Image assets

Place brand assets here for the public UI:

- **`logo.png`** — Used in the navbar (shown in a white rounded container). Prefer a version that reads well on light backgrounds; if your logo is dark-on-dark, consider a light or inverted variant.
- **`cover.png`** — Hero/cover image for the home page (campus, nature, or similar). Used as full-bleed background with a gradient overlay.

Images are referenced as `/images/logo.png` and `/images/cover.png`. If a file is missing, the app still works: the nav shows the text “Smart Trees” and the hero uses the gradient only.
